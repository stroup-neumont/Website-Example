Frequently Asked Questions

What topics does CSC210 cover?

CSC210 covers HTML5, CSS3, JavaScript, responsive design, web APIs, JSON/XML, and an introduction to modern frameworks.

Do I need prior programming experience?

While helpful, prior programming experience is not required. The course starts with fundamentals and builds up to more advanced concepts.

How can I prepare for CSC210?

Familiarize yourself with basic computer concepts and file management. Having a curious mindset and willingness to experiment with code is most important!

What software do I need for the course?

You'll need a modern web browser (Chrome, Firefox, or Edge) and a code editor like Visual Studio Code. All software used is free!
